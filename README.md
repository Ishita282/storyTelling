# Interactive Storytelling Project

## Overview
An interactive "choose-your-own-adventure" web application built using **HTML**, **CSS**, and **JavaScript**. Users make choices that branch the story in different directions, creating a unique and replayable experience.

## Features
- Dynamic story updates using JavaScript
- Multiple story paths and endings
- Responsive, animated interface
- Easy restart and navigation

## How to Run Locally

### 1. Install Node.js and Express:
   ```bash```
   npm install express

### 2. Start the server:
node server.js

### 3. Open your browser and go to:
http://localhost:3000


## Customization

Modify the story object in script.js to create your own adventure.

Add background images or audio for immersion.
